styleguide

documentation

Based on components from Semantic UI:

semantic-ui.com

Uses cdns for access but numerous recipes are available for integration:

semantic-ui.com/introduction/advanced-usage

styleguide version uses jquery and should not be used for angular/react projects

Dependencies:

-Semantic UI min.css

-jQuery min.js

-Material Icons

-Custom CSS (compiled from SASS)

-Semantic UI min.js
